## 📝 이용 규칙 및 팁

1. **노션 스타일 에디터**: 로컬 서버 실행 중에 상단의 **`✏️ 글쓰기`** 메뉴를 누르면 노션처럼 제목과 본문을 나누어 쉽게 글을 추가, 수정, 삭제할 수 있습니다.
2. **공동 작업**: 깃허브 저장소에 공동작업자로 등록된 멤버라면 누구나 새로운 문서를 추가하고 수정할 수 있습니다.
3. **수식 표기법**: 본문 중간에 수식을 적을 때는 인라인 형태 `$수식$`로 적고, 문단 전체에 크게 적을 때는 `$$수식$$` 블록 형태로 적으면 자동으로 예쁘게 그려집니다.
4. **실시간 배포**: 로컬 웹 에디터나 메모장으로 글을 수정한 뒤, 터미널에서 `git push`하여 코드를 올리면 1~2분 뒤 자동으로 인터넷 웹사이트에 업데이트됩니다.
